"SagaBlog Light Blog WordPress Theme" By WD village.

== Description ==
SagaBlog Light is responsive WordPress theme for personal blog. Added more then 30+ different home variations, 7 front page type, 5 widget areas, 11 custom widgets, page templates: 'page with contact info' and 'page with no sidebar', customizer for change  the theme to suit your desires. SagaBlog Light is translation ready, high resolution, ready for Contact Form 7, compatible with WooCommerce.
